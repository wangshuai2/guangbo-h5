## 📋 功能详情

### 手机号登录流程

```
1. 用户选择"手机登录" Tab
2. 输入11位手机号
3. 点击"获取验证码"按钮
4. 后端发送短信验证码
5. 用户输入6位验证码
6. 点击"登录"按钮
7. 登录成功 → 存储 token → 跳转首页
```

### 验证规则

| 字段   | 规则                     |
| ------ | ------------------------ |
| 手机号 | 11位，以1开头，第二位3-9 |
| 验证码 | 6位数字                  |

### 安全特性

- ✅ 手机号格式验证
- ✅ 验证码格式验证
- ✅ 60秒倒计时防止频繁发送
- ✅ 防重复点击
- ✅ 登录中 loading 状态
